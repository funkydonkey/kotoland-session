# Задание на реализацию сайта для семейной стратегической сессии

## Цель проекта
Создать интерактивный веб-сайт, который поможет проведению совместной семейной стратегической сессии для планирования 2026 года. Поможет сфокусироваться на общих ценностях, поставить совместные цели и придерживаться их в течении года. А также сделает сессию интереснее, повысит вовлеченность участников.

## Структура проекта
Проект должен состоять из нескольких блоков:
- Приглашение
- Рабочее пространство
- Итоги сессии

#### Приглашение
Простая веб-страничка описывающая цели и задачи сессии, а самое главное - рассказывающая как подготовиться к сессии:
``` txt 
За 3-5 дней до встречи каждому подготовить:
- Личный анализ года: главные достижения и сложности
- 3 категории желаний: карьера, семья, личное
- Финансовый обзор: основные траты за год
```

#### Рабочее пространство
Основная рабочая страница для проведении сессии. Страница состоящая из блоков описанных в файле structure.md
В каждом блоке есть примеры вопросов для обсуждение и одно большое текстовое поле, которое заполняют учатники. Всю внесенную информацию можно сохранить.

#### Итоги сессии
Страница, на которой отражаются обощенные ответы учатсников сохраненные в рабочем пространстве. В начале обозначены главные цели, а потом уже детализация.

## Технические требования
- Это не публичный проект, не нужна база данных и авторизация - все данные можно писать файл
- Деплой будет на GitHub Pages
- Если нужно используй AI агента для анализа полученных данных и генерации итоговых целей - можешь использовать OpenAI Agents SDK - ключ от OpenAI API лежат в .env файле
